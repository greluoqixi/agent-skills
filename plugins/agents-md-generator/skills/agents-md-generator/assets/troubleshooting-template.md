# Troubleshooting & Fix Knowledge

> Time decay: prefer records from the last 6 months. Entries older than 6 months are tagged `[may be stale]`.

## Record Format

Each record has two parts:

### Retrospect
- **Symptoms**: Error behavior (logs, error messages, unexpected output)
- **Root cause**: The chain of causes that led to the problem
- **Solution**: Steps taken to fix it, with key code snippets or file paths

### Prevention
- **Mistake pattern**: Why AI initially went in the wrong direction (misjudgment, omission, wrong assumption)
- **Check rule**: A condition to verify before making similar changes (one executable check item)
- **Related modules**: Upstream/downstream files to review when modifying this kind of code
